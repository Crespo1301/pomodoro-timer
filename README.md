# üçÖ Pomodoro Timer CLI

A simple command-line Pomodoro timer with session tracking.

## Features

- 25-minute work sessions / 5-minute breaks
- Live countdown with progress bar
- Session tracking (saved to JSON)
- Stats: sessions today and this week

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/pomodoro-timer.git
cd pomodoro-timer
python pomodoro.py
```

## Usage
```
Options:
  [s] Start Pomodoro (25 min work + 5 min break)
  [w] Work session only (25 min)
  [b] Break only (5 min)
  [t] View stats
  [q] Quit
```
